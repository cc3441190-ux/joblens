三步上手 joblens

从加载扩展到在 Boss 上点选岗位，整条路径尽量短。

步骤 01

加载扩展

在 Chrome 打开扩展管理页，开启开发者模式后「加载已解压」，选中包含 manifest.json 的文件夹。

步骤 02

侧栏完成向导

按需配置 API Key、上传或粘贴简历，完成简历画像与分析，确认进入求职模式。

步骤 03

在 Boss 点选岗位

列表中单击职位卡片，侧栏自动对齐当前 JD，查看透视要点与适配建议。

为什么侧栏更合适

信息密度收在助手面板里，Boss 列表仍是你的主工作台。

推荐使用最新稳定版 Chrome。当前产品聚焦 Boss 直聘 Web 端（岗位列表单击卡片）；其他站点未做适配承诺。
